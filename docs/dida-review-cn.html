为何下滑 — 假设

DIDA 自身原因（同业均在增长）

既然大盘与同业都在增长，原因就出在DIDA 与 OMH 之间。优先假设：

1
Single Use 房型映射问题 高度可能

DIDA 持续存在 Single Use 房型命名（roomTypeCode）问题。映射错乱会减少可售库存 → 搜索与转化下降。（内部 T19 处理中）

2
价格 / 覆盖竞争力不足 高度可能

在同一库存上，Convergent 与 PKFARE 以更优的价格/覆盖胜出，吸收了 DIDA 的量。艺龙也同步下滑（−34%），呈现"中国系低毛利"竞争挤压。

3
QPS / 搜索覆盖受限

DIDA 曾有 QPS 受限历史。每秒请求受限会结构性降低搜索曝光与转化。

4
L2B（搜索→预订）转化下降

DIDA 前端曝光/转化逻辑可能有变化，需 DIDA 内部指标确认。

⚠️ 本次数据中营收与 ADR 可靠性不足（导出文件的 revenue 字段问题），因此仅采用量的指标（预订量、间夜）。结论（量 −43%、份额腰斩、落后同业）在量口径上是稳健的。
